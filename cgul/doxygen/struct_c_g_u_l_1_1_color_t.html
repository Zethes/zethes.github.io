template<typename VectorType >
CGUL::Vector3T< VectorType > CGUL::ColorT< Type >::ToVector3 ( ) const
inline

Calculates RGB values from 0.0f to 1.0f and returns a Vector3 with the result.

Each element is calculated by taking their respective values and dividing by MAX. This is useful when colors need to be represented as floating point numbers rather than byte values, as is the case in a lot of API's (especially graphical ones, such as OpenGL).

Returns
A vector containing the r, g, b values as floating point x, y, z.
