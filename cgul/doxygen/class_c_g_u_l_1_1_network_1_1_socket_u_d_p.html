void CGUL::Network::SocketUDP::Bind ( unsigned short  port,
bool  ipv4 = true 
)

Binds the socket to a specified port.

Since UDP is connectionless, all that is required for a connection is a port. To send to a specific host use SendTo. When receiving, use ReceiveFrom.

Parameters
portThe port to receive packets on.
ipv4True for IPv4, false for IPv6.
void CGUL::Network::SocketUDP::Close ( )

Gracefully closes the connection.

Note
Because UDP sockets are connectionless, no remote hosts will be notified when this socket is closed.

unsigned short  port

Binds the socket to a specified port, and restricts communication to one IP address.

Although UDP is a connectionless protocol, it is still possible to "connect" to a single host. After connecting, use Send and Receive methods instead of SendTo and ReceiveFrom.

Parameters
ipThe host to send all packets to.
