CGUL::Network::IPAddress::IPAddress ( const CGUL::String &  ip)

Parses a string creating an IPv4 address, an IPv6 address, or an invalid address.

This constructor can be used to convert an unknown IP address into an IPv4 address or an IPv6 address. First it attempts to convert the string in IPv4, and if that fails it attempts to convert to an IPv6. If neither conversion succeeds, the resulting address is invalid.

Parameters
ipA parsable ip address, either IPv4 or IPv6.

Calculates the broadcast address of a network given an address and netmask.

The broadcast address can only be calculated on a IPv4 network. While it is technically possible to calculate an IPv6 broadcast address, the IPv6 protocol has dropped support for broadcasting. Since broadcasting is not supported in IPv6, this method will result in an exception given an IPv6 address.

Parameters

CGUL::String CGUL::Network::IPAddress::ToString ( ) const

Attempts to convert the IP address to a human readable string.

The conversion is based on the current IP address type (as returned by GetType). This method uses internal network functions to do the conversion, and upon failure will return a string of "Invalid" followed by the IP address type. If the IP address type is IPAddressType::INVALID then this method simply returns "Invalid."

Returns
A string of the human readable IP address.
void CGUL::Network::IPAddress::ToUInt128 ( UInt64 *  in) const

Gets a 128 bit unsigned integer from the address defined as an array of two 64 bit unsigned integer values.

This method will reliably capture an IPv4 or IPv6 address as a 128 bit unsigned integer value. Since an IPv4 address is only 32 bits, the unused bits will be zerod out.

Parameters
inAn array of two 64 bit unsigned integers to store the address into.
CGUL::UInt32 CGUL::Network::IPAddress::ToUInt32 ( ) const

Gets a 32 bit unsigned integer from the address, only valid for IPv4 addresses.

While it is possible to call this method on an IPv6 address, the resulting value will be unreliable since it will only contain part of the actual address. In the off chance that the address may be IPv4 or IPv6, consider using ToUInt128 instead.

Returns
A 32 bit unsigned integer.
